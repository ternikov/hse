Machine Learning

Machine Learning (ML) is about extracting knowledge from data. It is a research field at the intersection of statistics, artificial intelligence, and computer science and is also known as predictive analytics or statistical learning.

There are three most regularly listed categories of ML:

  • Supervised Learning
  • Unsupervised Learning
  • Reinforcement Learning

Machine Learning

Source: http://www.cognub.com/index.php/cognitive-platform/

Supervised Learning

Problems to be solved:

  • Classification process of assigning category to input data sample
  • Regression process of predicting a continuous, numerical value for input data sample

Examples:

  • predicting whether a person is ill or not
  • detecting fraudulent transactions

  • identifying the zip code from handwritten digits on an envelope

  • assessing the house price
  • forecasting grocery store food demand

  • temperature forecasting

Supervised Learning

Source: https://towardsdatascience.com/

Unsupervised Learning

Group of algorithms that try to draw inferences from non-labeled data

Problems to be solved:

  • Clustering (pattern recognition) - process of dividing and grouping similar data samples together. Groups are usually called clusters
  • Association & Dimensionality reduction - process of compressing features (input variables used in making predictions) into so-called principal values which conveys similar information concisely

Examples:

  • identifying topics in a set of blog posts
  • segmenting customers into groups with similar preferences
  • Airbnb groups its housing listings into neighborhoods so that users can navigate listings more easily
